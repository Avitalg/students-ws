Welcome to My API.

This data could help you use my API.

Get All Students

If you want to get all student, please enter the following URL: http://localhost:3000/allStudents

Result example: [ { student_id: 1, name: "tom", area: "center", degree: "bsc", grade: 81, year: 1 }, { student_id: 2, name: "yossi", area: "south", degree: "ba", grade: 94, year: 2 }, { student_id: 3, name: "bar", area: "north", degree: "bsc", grade: 92, year: 1 }, { student_id: 4, name: "gal", area: "north", degree: "ba", grade: 72, year: 3 }, ]

Get Student By Id

you can find a student by his Id, by entering the following URL: http://localhost:3000/getStudent/id

Example: http://localhost:3000/getStudent/1 and the result will be { student_id: 1, name: "tom", area: "center", degree: "bsc", grade: 81, year: 1 }

get All Excellence Students

you can find all excellence students, by entering the following URL: http://localhost:3000/getAllExcellenceStudents
